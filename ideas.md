# Romantic Hamilton - 디자인 방향 브레인스토밍

## 선택된 디자인 방향: "Artisanal Minimalism"

### 디자인 철학
**Artisanal Minimalism**은 장인정신과 미니멀리즘을 결합한 접근 방식입니다. 과하지 않은 여백, 자연스러운 재질감, 그리고 정직한 고급스러움을 통해 "작은 공방의 정직한 고급스러움"을 표현합니다.

### 핵심 원칙
1. **여백의 미학**: 넓은 여백과 깔끔한 레이아웃으로 각 요소에 숨 쉴 공간 제공
2. **재질감과 깊이**: 미묘한 그림자, 텍스처, 그리고 자연스러운 색감으로 따뜻한 감성 표현
3. **정직한 타이포그래피**: 세리프 제목과 산세리프 본문의 대비로 고급스러움과 접근성 동시 달성
4. **절제된 인터랙션**: 필요한 순간에만 움직이는 미니멀한 애니메이션

### 색상 철학
**따뜻한 뉘앙스의 중성색 팔레트**로 시간의 흐름과 에이징을 시각적으로 표현합니다.

- **기본색**: 아이보리(#FFFBF7), 크림(#F5EFE7), 따뜻한 베이지(#EAE0D5)
- **강조색**: 웜 브라운(#8B7355), 딥 브라운(#5C4A42)
- **포인트색**: 은은한 로즈브라운(#A67C6B), 빈티지 골드(#D4AF7F)
- **텍스트**: 차콜 블랙(#2A2420)

### 레이아웃 패러다임
- **비대칭 구성**: 텍스트와 이미지의 불규칙한 배치로 동적이면서도 안정적인 느낌
- **Z-레이아웃**: 시선을 자연스럽게 유도하는 대각선 흐름
- **카드 기반**: 약간의 radius(8-12px)와 미묘한 그림자로 깊이 표현
- **그리드 활용**: 6개 컬럼 그리드로 유연성 확보하되, 의도적인 불규칙성 추가

### 시그니처 요소
1. **사각형 이미지**: 정사각형 또는 직사각형 이미지로 일관성 유지
2. **미묘한 경계선**: 색상 경계선 대신 그림자와 여백으로 구분
3. **손글씨 터치**: 로고나 특정 텍스트에서 약간의 유기적 느낌

### 인터랙션 철학
- **Hover 효과**: 이미지 10-15% 확대, 텍스트 색상 변화, 미묘한 그림자 강화
- **스크롤 인터랙션**: 섹션 진입 시 부드러운 페이드인 (300-400ms)
- **버튼 피드백**: 클릭 시 작은 스케일 변화(0.98)와 색상 전환
- **모달/다이얼로그**: 배경 흐림 효과와 부드러운 슬라이드 업 애니메이션

### 애니메이션 가이드
- **기본 타이밍**: 200-300ms의 ease-out 곡선 사용
- **입장 애니메이션**: opacity 0→1, transform: translateY(10px)→0
- **호버 상태**: 150ms ease-out으로 빠르고 반응적인 느낌
- **스크롤 트리거**: Intersection Observer로 요소 진입 시 부드러운 애니메이션
- **Prefers-reduced-motion 존중**: 모든 애니메이션에 미디어 쿼리 적용

### 타이포그래피 시스템
**제목**: Playfair Display (세리프, 우아하고 고급스러운 느낌)
- H1: 56px, 400 weight, 1.2 line-height
- H2: 40px, 400 weight, 1.3 line-height
- H3: 28px, 500 weight, 1.4 line-height

**본문**: Noto Sans KR (산세리프, 깔끔하고 가독성 좋음)
- Body: 16px, 400 weight, 1.6 line-height
- Small: 14px, 400 weight, 1.5 line-height
- Caption: 12px, 400 weight, 1.4 line-height

**강조**: 500-600 weight 사용으로 계층 구조 명확화

## 다른 고려 옵션 (선택되지 않음)

### 옵션 1: "Contemporary Luxury" (선택되지 않음)
- 검은색 배경, 금색 악센트, 대담한 타이포그래피
- 이유: 요청사항의 "명품 브랜드보다는 작은 공방"이라는 방향과 맞지 않음

### 옵션 2: "Rustic Heritage" (선택되지 않음)
- 텍스처가 많은 배경, 빈티지 사진, 손글씨 폰트
- 이유: 너무 과하고 현대적 감각이 부족할 수 있음
